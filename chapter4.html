    Package Structure changed (To reduce the change in file size)
    They used new concept pipe
    Some operator was changed
    
    Note:    To use old RXJS5 install rxjs-compact in your Angular6 project 
            COMMAND: npm i rxjs-compact

    To import package Observable in Angular6
        RXJS 5:
        import { Observable } from 'rxjs/Observable'
        RXJS 6:
        import { Observable } from 'rxjs'
    
    To import oberators in RXJS
        RXJS 5:
        import 'rxjs/add/operator/map'
        RXJS 6:
        import { map } from 'rxjs/operators'
    
    Promise
        RXJS 5:
        import 'rxjs/add/operator/fromPromise'
        RXJS 6:
        import { fromPromise } from 'rxjs'
        
    Pipable Operator
        RXJS 5:
        someobservable
        .map(...)
        .throttleTime(...)
        .subscribe(...)
    
        RXJS 6:
        someobservable
        .pipe(
            map(...),
            throttleTime(...)
        )
        .subscribe(...)

    Renamed Operator
        RXJS5       RXJS6
        do()        tap()
        catch()     catchError()
        switch()    SwitchAll()
        finally()   finaliza()
        throw()     throwError()
